Introduction to Modern Layout

Modern CSS layout techniques have revolutionized the way we structure web pages.

Before Flexbox and Grid, developers relied heavily on floats, positioning, and hacks.

div { float: left; }

Flexbox and Grid provide more intuitive and powerful systems for layout.

This documentation will guide you through their basics and how to apply them.

Why Flexbox & Grid Matter

These layout models are responsive-friendly and easier to read and maintain.

Flexbox excels at 1D layout (rows or columns), Grid at 2D (rows and columns).

  • Cleaner code
  • Fewer media queries needed
  • Better alignment and distribution

Modern browsers have excellent support for both.

display: flex;
Flexbox Basics & Use Cases

Flexbox makes it easy to distribute space and align items in a container.

Useful for navbars, cards, layouts with equal heights.

display: flex; justify-content: space-between;

Use align-items for vertical alignment.

Use flex-wrap to control item wrapping.

CSS Grid Overview

CSS Grid is perfect for creating two-dimensional layouts with ease.

Grids define rows and columns simultaneously.

display: grid; grid-template-columns: 1fr 1fr;

You can place elements with grid-column and grid-row.

  • Repeatable rows and columns
  • Named areas for readability
Responsive Design Strategies

Media queries adapt layout to different screen sizes.

Use @media to define breakpoints.

@media (max-width: 768px) { ... }

Flexbox and Grid both respond well to these techniques.

  • Stack content vertically on mobile
  • Change grid structure at breakpoints
